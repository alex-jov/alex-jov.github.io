SAE 1.03 - Transmission Systems Characterization
Connect

This SAE project involved characterizing transmission systems used in computer networks, particularly Ethernet and coaxial cables. The objective was to understand and quantify the two main parameters: attenuation and delay.

The project was conducted through three complementary approaches: theoretical study using mathematical equations, computer simulation with specialized software, and preparation for real measurements.

Skills developed: Characterization of physical links (copper, fiber optic), Attenuation and delay measurement, Transmission system simulation, Ethernet and coaxial cable analysis

Technologies used: LTspice, MATLAB

Context: SAE 1.03 - IUT de Béthune - January 2025

Local AI Video Upscaling
Program

A fully local AI video enhancement tool that runs entirely offline, without cloud services or API keys. The program uses Real-ESRGAN to increase video resolution while preserving the original audio quality.

The process consists of three steps: extracting frames from the video file using FFmpeg, enhancing each frame through the Real-ESRGAN AI model, and then reconstructing the video with the upscaled frames. The graphical interface developed with Tkinter makes the tool accessible and easy to use.

Technologies used: Python 3.9+, Real-ESRGAN (NCNN Vulkan), FFmpeg, FFprobe, Tkinter, Vulkan GPU Acceleration

Supported formats: MP4, MKV, AVI, MOV

Context: Personal project
